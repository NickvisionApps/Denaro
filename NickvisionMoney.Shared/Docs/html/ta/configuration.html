உள்ளமைவு

பயன்பாட்டு உள்ளமைவில் நீங்கள் எதை மாற்றலாம் என்பதை இந்த பக்கம் விவரிக்கிறது.

உங்கள் தளத்தைப் பொறுத்து, பயன்பாட்டு இடைமுகத்தில் உள்ளமைவு விருப்பத்தேர்வுகள் அல்லது அமைப்புகள் என குறிப்பிடப்படுகிறது.

கருப்பொருள்

ஒளி அல்லது இருண்ட கருப்பொருளை அமைக்கவும் அல்லது டெனாரோ உங்கள் கணினி கருப்பொருளைப் பின்பற்றவும். சாளரங்களில் இதை மாற்றுவதற்கு விண்ணப்பிக்க விண்ணப்பிக்க வேண்டும்.

பரிவர்த்தனை இயல்புநிலை நிறம்

A color that will be selected by default when adding a new transaction with unique color. Changing this will not affect existing transactions, even if they use previously selected default color.

இயல்புநிலை நிறத்தை மாற்றவும்

பரிமாற்றம் ஐப் பயன்படுத்தி உருவாக்கப்பட்ட பரிவர்த்தனைகளுக்கு பயன்படுத்தப்படும் வண்ணம். இதை மாற்றுவது தற்போதுள்ள பரிவர்த்தனைகளை பாதிக்காது.

குழு இயல்புநிலை நிறம்

புதிய குழுவைச் சேர்க்கும்போது இயல்பாகவே தேர்ந்தெடுக்கப்படும் வண்ணம். இது «ungrouped» குழுவிற்கும் ஒரு வண்ணம்.

கணக்கு நிறத்தை சரிபார்க்கிறது

A color used to mark accounts with the Checking type in a recent accounts list.

சேமிப்பு கணக்கு நிறம்

A color used to mark accounts with the Savings type in a recent accounts list.

வணிக கணக்கு நிறம்

A color used to mark accounts with the Business type in a recent accounts list.

சொந்த இலக்கங்களைப் பயன்படுத்துங்கள்

லத்தீன் இலக்கங்களுக்கு பதிலாக உங்கள் இருப்பிடத்திற்கு சொந்தமான எண்களைப் பயன்படுத்த வேண்டுமா. எடுத்துக்காட்டாக, இந்த அமைப்பு இயக்கப்பட்டிருந்தால் ٠١٢ கிழக்கு அரபு எண்களைப் பயன்படுத்தும் இடங்களுக்கு 012 க்கு பதிலாக பயன்படுத்தப்படும்.

தசம பிரிப்பான் செருகவும்

எந்த விசைகள் அழுத்தின. தசம பிரிப்பான் ஏற்கனவே ஒரு துறையில் வழங்கப்பட்டால், அது செருகப்படாது.

காபிம காப்புப்பிரதி கோப்புறை

A folder where your data will be automatically exported to CSV after every change. This feature doesn't work for password-protected accounts, because CSV files can't be password-protected.
